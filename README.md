# <a name="microsoft-graph-documentation"></a>Документация по Microsoft Graph

Благодарим вас за интерес к документации по Microsoft Graph! Лучше всего просматривать эти материалы на [Портале разработчиков Microsoft Graph](https://developer.microsoft.com/ru-RU/graph/docs/concepts/overview).

## <a name="give-us-your-feedback"></a>Оставьте свой отзыв

Ваш отзыв важен для нас.

- Если у вас возникли вопросы или вы нашли ошибки в документации, [отправьте запрос](https://github.com/microsoftgraph/microsoft-graph-docs/issues) касательно этого репозитория.
- Мы также рекомендуем создать вилку проекта, внести исправление и отправить запрос на вытягивание. Чтобы добавить рекомендации, см. статью [CONTRIBUTING.md](CONTRIBUTING.md).
- Чтобы рассказать о своих впечатлениях от использования файлов и пожеланиях насчет будущих версий, примеров кода и т. д., поделитесь своими мыслями на [сайте UserVoice платформы разработки для Office](https://officespdev.uservoice.com/forums/224641-feature-requests-and-feedback?category_id=101632).
