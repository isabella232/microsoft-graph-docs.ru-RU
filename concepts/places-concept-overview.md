# <a name="places-api-overview-preview"></a>Обзор API мест (предварительная версия)

Вы можете использовать API мест для создания привлекательных приложений, учитывающих различные расположения. Теперь вы можете получать доступ к местам компании и сведениям о каждом из них.

<!-- markdownlint-disable MD026 -->
## <a name="why-integrate-with-places"></a>Преимущества интеграции с местами
<!-- markdownlint-enable MD026 -->

Современные приложения обычно учитывают расположение, чтобы предоставлять интеллектуальный контекст. Это помогает пользователю находить место или позволяет приложению учитывать расположение пользователя в рамках своего решения. [API мест](/graph/api/resources/place?view=graph-rest-beta) предоставляет подробные сведения о расположениях, например помещения или списки помещений, для реализации этих сценариев.

### <a name="help-customers-stay-synchronized-and-navigate-their-day"></a>Помогите клиентам поддерживать синхронизацию и ориентироваться в рабочем дне

Добавьте расположения в календарь и помогите пользователям ориентироваться в их рабочем дне и повышать эффективность работы. Включите автоматических помощников, чтобы предоставлять сведения о помещениях в здании. Отправляйте по электронной почте дополнительные сведения своим гостям с указанием маршрута к помещению на карте. Создавайте собственные приложения для организации питания на мероприятиях и используйте [API мест](/graph/api/resources/place?view=graph-rest-beta) для настройки помещений.
