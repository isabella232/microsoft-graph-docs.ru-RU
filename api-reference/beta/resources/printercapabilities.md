# <a name="printercapabilities-complex-type"></a>сложный тип Принтеркапабилитиес

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет сведения о возможностях принтера.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|исколорпринтингсуппортед|Boolean|True, если принтер поддерживает цветную печать; в противном случае — false. Только для чтения.|
|суппортсфитпдфтопаже|Boolean|Имеет значение true, если принтер поддерживает масштабирование PDF-страниц для согласования с размером носителя печати; в противном случае — false.|
|суппортедкопиеспержоб|[интежерранже](integerrange.md)|Диапазон копий для каждого задания, поддерживаемый принтером.|
|суппортеддокументмиметипес|Коллекция строк|Типы MIME документов, поддерживаемые принтером.|
|суппортедфинишингс|Коллекция Принтфинишинг|Окончания, поддерживаемые принтером.|
|суппортедмедиаколорс|Коллекция строк|Цвета мультимедиа (например, бумаги), поддерживаемые принтером.|
|суппортедмедиатипес|Коллекция Принтмедиатипе|Типы мультимедиа, поддерживаемые принтером. Допустимые значения описаны в приведенной ниже таблице.|
|суппортеддуплексконфигуратионс|Коллекция Принтдуплексконфигуратион|Дуплексные конфигурации, поддерживаемые принтером. Допустимые значения описаны в приведенной ниже таблице.|
|суппортедмедиасизес|Коллекция строк|Размеры носителей, поддерживаемые принтером. Поддерживает стандартные имена размеров для форматов файлов ISO и ANSI, а также произвольные размеры, поддерживаемые соответствующим принтером.|
|суппортедпажеспершит|[интежерранже](integerrange.md)|Значения Пажеспершит, поддерживаемые принтером.|
|суппортедориентатионс|Коллекция Принториентатион|Ориентация печати, поддерживаемые принтером. Допустимые значения описаны в приведенной ниже таблице.|
|суппортедаутпутбинс|Коллекция строк|Поддерживаемые выходные лотки принтера (лотки).|
|суппортедпресентатиондиректионс|Коллекция Принтпресентатиондиректион|Направления презентации, поддерживаемые принтером. Поддерживаемые значения описаны в приведенной ниже таблице.|
|суппортедколорконфигуратионс|Коллекция Принтколорконфигуратион|Цветовые режимы, поддерживаемые принтером. Допустимые значения описаны в приведенной ниже таблице.|
|суппортедпринткуалитиес|Коллекция Принткуалити|Качество печати, поддерживаемое принтером.|

|:---|:---|:---|

|каунтерклокквисефромтоплефт|1,1|Расположите страницы в сетке против часовой стрелки, начиная с верхнего левого угла.|
|каунтерклокквисефромтопригхт|2|Расположите страницы в сетке против часовой стрелки, начиная с верхнего правого.|
|клокквисефромтопригхт|4|Расположите страницы в сетке по часовой стрелке, начиная с верхнего правого.|
|каунтерклокквисефромботтомлефт|4 |Расположите страницы в сетке против часовой стрелки, начиная с левой нижней части.|
|клокквисефромботтомлефт|5 |Расположите страницы в сетке по часовой стрелке, начиная с левой нижней части.|
|каунтерклокквисефромботтомригхт|6 |Расположите страницы в сетке против часовой стрелки, начиная с правого нижнего края.|
|клокквисефромботтомригхт|7 |Расположите страницы в сетке по часовой стрелке, начиная с правого нижнего края.|

### <a name="printduplexconfiguration-values"></a>значения Принтдуплексконфигуратион

|Элемент|Значение|Описание|
|:---|:---|:---|
|твосидедлонжедже|нуль|Принтер будет печататься на двух сторонах и будет переворачивать документы вдоль длинного края.|
|твосидедшортедже|1,1|Принтер будет печататься на двух сторонах и будет переражать документы вдоль короткого края.|
|онесидед|2|На принтере будет напечатана только одна сторона.|

## <a name="printorientation-values"></a>значения Принториентатион

|Элемент|Значение|Описание|
|:---|:---|:---|
|ориентацию|4|Принтер будет печатать впечатления в "книжной" ориентации.|
|вдоль|4 |Принтер будет печатать впечатления на "альбомной" ориентации.|
|реверселандскапе|5 |Принтер будет печатать впечатления на ориентации "Обратная ориентация".|
|реверсепортраит|6 |Принтер будет печатать впечатления на ориентации "Обратная книжная".|

### <a name="printcolorconfiguration-values"></a>значения Принтколорконфигуратион

|Элемент|Значение|Описание|
|:---|:---|:---|
|блаккандвхите|нуль|Черно-белое (используйте только черные материалы с маркерами).|
|черн|1,1|Оттенки серого (могут использоваться некоторые материалы цветового маркера).|
|color|2|Color (Используйте любую комбинацию материалов маркера, чтобы создать впечатление цвета).|
|Авто|4|Разрешите принтеру выбрать используемый цветовой режим.|
