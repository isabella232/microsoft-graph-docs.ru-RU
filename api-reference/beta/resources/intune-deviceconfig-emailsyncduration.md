# <a name="emailsyncduration-enum-type"></a>Тип перечисления emailSyncDuration

> **Важные:** Интерфейсы API в разделе версии /beta в Microsoft Graph могут быть изменены. Использование этих API в производственных приложениях не поддерживается.

> **Примечание:** Microsoft Graph API для Intune требуется [Активная лицензия Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Возможные значения для продолжительность синхронизации электронной почты.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|userDefined|0|User Defined, значение по умолчанию, без цели.|
|oneDay|1|Синхронизация электронной почты на один день.|
|threeDays|2|Синхронизация электронной почты на три дня.|
|oneWeek|3|Синхронизация неделю сообщений электронной почты.|
|twoWeeks|4|Синхронизация две недели по электронной почте.|
|oneMonth|5|Синхронизация месяц сообщений электронной почты.|
|без ограничений|6|Синхронизируйте неограниченное время сообщений электронной почты.|
