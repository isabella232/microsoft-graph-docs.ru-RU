# <a name="emailsyncduration-enum-type"></a>тип перечисления Емаилсинкдуратион

Пространство имен: Microsoft. Graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Возможные значения срока действия синхронизации электронной почты.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|UserDefined типа|нуль|Пользователь определен, значение по умолчанию, без намерения.|
|онедай|1 |Синхронизация одного дня электронной почты.|
|сридайс|2 |Синхронизация трех дней электронной почты.|
|оневик|3 |Синхронизация одной недели с электронной почтой.|
|твовикс|4 |Синхронизируйте две недели с электронной почтой.|
|онемонс|5 |Синхронизируйте один месяц электронной почты.|
|неограниченно|6 |Синхронизация неограниченной длительности электронной почты.|
