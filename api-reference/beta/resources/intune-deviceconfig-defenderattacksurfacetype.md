> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Возможные значения правил уменьшения зоны атак защитника

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|UserDefined типа|нуль|Значение по умолчанию, которое отключает правило сокращения для уязвимой зоны.|
|блок|1,1|Включите правило сокращения для уязвимой зоны.|
|аудитмоде|2|Оцените, как правило ASR будет влиять на вашу организацию, если она включена. Не изменяет функциональность, но создает журналы.|
|пользователя|6 |Предупреждающее сообщение о том, что пользователь может обходить блок из правила уменьшения уязвимой зоны.|
