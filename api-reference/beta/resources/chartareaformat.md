# <a name="chartareaformat-resource-type"></a>Тип ресурса ChartAreaFormat

> **Важно!** API бета-версии (/beta) в Microsoft Graph проходят тестирование и могут быть изменены. Использование этих API в производственных приложениях не поддерживается.

Инкапсулирует свойства формата для всей области диаграммы.


## <a name="methods"></a>Методы
Нет

## <a name="properties"></a>Свойства
Нет

## <a name="relationships"></a>Связи
| Связь | Тип   |Описание|
|:---------------|:--------|:----------|
|fill|[ChartFill](chartfill.md)|Представляет формат заливки объекта, включая сведения о форматировании фона. Только для чтения.|
|font|[ChartFont](chartfont.md)|Представляет атрибуты шрифта (имя, размер, цвет и т. д.) для текущего объекта. Только для чтения.|
