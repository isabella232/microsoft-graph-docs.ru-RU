# <a name="printerdefaults-resource-type"></a>Тип ресурса Принтердефаултс

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет параметры принтера по умолчанию. Проверьте [возможности](../api/printer-getcapabilities.md) принтера, чтобы увидеть все поддерживаемые им значения.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|копиеспержоб|Int32|Число копий по умолчанию, печатаемых для каждого задания.|
|документмиметипе|String|Используемый по умолчанию тип MIME для обработки документов.|
|завершения|Коллекция Принтфинишинг|Набор окончаний по умолчанию, применяемых к заданиям печати. Допустимые значения описаны в приведенной ниже таблице.|
|медиаколор|String|Используемый по умолчанию носитель (например, бумага) для печати документа.
|mediaType|принтмедиатипе|Используемый по умолчанию носитель (например, бумага) для печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|медиасизе|String|Используемый по умолчанию размер носителя. Поддерживает стандартные имена размеров для форматов файлов ISO и ANSI, а также произвольные размеры, поддерживаемые соответствующим принтером.
|пажеспершит|Int32|Число страниц документов по умолчанию для печати на каждом листе.
|orientation|принториентатион|Ориентация по умолчанию, используемая при печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|аутпутбин|String|Выходной лоток по умолчанию для размещения завершенных отпечатков. Узнайте о [возможностях](../api/printer-getcapabilities.md) принтера для списка поддерживаемых выходных лотков.|
|пдффиттопаже|Boolean|Параметр Пдффиттопаже по умолчанию. Значение true, чтобы разместить каждую страницу документа PDF на физическом листе мультимедиа; false, чтобы разрешить принтеру определять способ макетирования просмотров.|
|пресентатиондиректион|принтпресентатиондиректион|Направление по умолчанию для размещения страниц при печати нескольких страниц на листе. Допустимые значения описаны в приведенной ниже таблице.|
|принтколорконфигуратион|принтколорконфигуратион|Цветовой режим по умолчанию, используемый при печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|принткуалити|принткуалити|Качество, используемое по умолчанию при печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|дуплексконфигуратион|принтдуплексконфигуратион|Дуплексная (двухсторонняя) конфигурация, используемая по умолчанию при печати документа. Допустимые значения описаны в приведенной ниже таблице.|

### <a name="printpresentationdirection-values"></a>значения Принтпресентатиондиректион

|Элемент|Значение|Описание|
|:---|:---|:---|
|клокквисефромтоплефт|нуль|Расположите страницы в сетке по часовой стрелке, начиная с верхнего левого угла.|
|каунтерклокквисефромтоплефт|1,1|Расположите страницы в сетке против часовой стрелки, начиная с верхнего левого угла.|
|каунтерклокквисефромтопригхт|2|Расположите страницы в сетке против часовой стрелки, начиная с верхнего правого.|
|клокквисефромтопригхт|4|Расположите страницы в сетке по часовой стрелке, начиная с верхнего правого.|
|каунтерклокквисефромботтомлефт|4 |Расположите страницы в сетке против часовой стрелки, начиная с левой нижней части.|
|клокквисефромботтомлефт|5 |Расположите страницы в сетке по часовой стрелке, начиная с левой нижней части.|
|каунтерклокквисефромботтомригхт|6 |Расположите страницы в сетке против часовой стрелки, начиная с правого нижнего края.|
|клокквисефромботтомригхт|7 |Расположите страницы в сетке по часовой стрелке, начиная с правого нижнего края.|

### <a name="printduplexconfiguration-values"></a>значения Принтдуплексконфигуратион

|Элемент|Значение|Описание|
|:---|:---|:---|
|твосидедлонжедже|нуль|Принтер будет печататься на двух сторонах и будет переворачивать документы вдоль длинного края.|
|твосидедшортедже|1,1|Принтер будет печататься на двух сторонах и будет переражать документы вдоль короткого края.|
|онесидед|2|На принтере будет напечатана только одна сторона.|

|еджеститчботтом|27|Пограничный — сшивка документа вдоль нижнего края.|

|стапледуалтоп|суммируемых|Сшивать документ дважды вдоль верхнего края.|
|стапледуалригхт|более|Сшивать документ дважды по правому краю.|
|стапледуалботтом|длиной|Сшивать документ дважды вдоль нижнего края.|

## <a name="printorientation-values"></a>значения Принториентатион

|Элемент|Значение|Описание|
|:---|:---|:---|
|ориентацию|4|Принтер будет печатать впечатления в "книжной" ориентации.|
|вдоль|4 |Принтер будет печатать впечатления на "альбомной" ориентации.|
|реверселандскапе|5 |Принтер будет печатать впечатления на ориентации "Обратная ориентация".|
|реверсепортраит|6 |Принтер будет печатать впечатления на ориентации "Обратная книжная".|

### <a name="printmediatype-values"></a>значения Принтмедиатипе

|Элемент|Значение|Описание|
|:---|:---|:---|
|Бланка|нуль|Стандартные листы бумаги.|
|кладка|1,1|Пленка для прозрачности, используемая с проекторами.|
|баллон|2|Конверт.|
|енвелопеплаин|4|Обычный конверт.|
|постоянных|4 |Непрерывный рулон бумаги.|
|странице|5 |Цифровой экран.|
|скринпажед|6 |Цифровой экран с поддержкой разбиения по страницам.|
|континуауслонг|7 |Длительный непрерывный рулон бумаги.|
|континуаусшорт|8 |Короткий непрерывный рулон бумаги.|
|енвелопевиндов|9 |Конверт с прозрачным окном, которое вырезается.|
|мултипартформ|10 |Форма из нескольких частей.|
|Слой|11 |Многоуровневая среда.|
|Метка|12 |Лист с обрезкоми подписей.|

### <a name="printquality-values"></a>значения Принткуалити

|Member|Описание|
|:---|:---|
|потребление|Принтер выполнит печать задания, используя низкую (обычно называемую "черновой") качеством.|
|medium|Принтер выполнит печать задания, используя медим (обычно известное как "нормальное").|
|высокоуровневых|Принтер выполнит печать задания, используя высокое (обычное, известное как "наилучшее" или "точное") качество.|

### <a name="printcolorconfiguration-values"></a>значения Принтколорконфигуратион

|Элемент|Значение|Описание|
|:---|:---|:---|
|блаккандвхите|нуль|Черно-белое (используйте только черные материалы с маркерами).|
|черн|1,1|Оттенки серого (могут использоваться некоторые материалы цветового маркера).|
|color|2|Color (Используйте любую комбинацию материалов маркера, чтобы создать впечатление цвета).|
|Авто|4|Разрешите принтеру выбрать используемый цветовой режим.|
