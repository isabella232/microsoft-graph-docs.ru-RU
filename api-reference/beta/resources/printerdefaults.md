# <a name="printerdefaults-resource-type"></a>Тип ресурса Принтердефаултс

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет параметры принтера по умолчанию. Проверьте [возможности](../api/printer-getcapabilities.md) принтера, чтобы увидеть все поддерживаемые им значения.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|копиеспержоб|Int32|Число копий по умолчанию, печатаемых для каждого задания.|
|contentType|String|Тип контента по умолчанию (MIME), используемый при обработке документов.|
|finishings|Коллекция Принтфинишинг|Набор окончаний по умолчанию, применяемых к заданиям печати. Допустимые значения описаны в приведенной ниже таблице.|
|медиаколор|Строка|Используемый по умолчанию носитель (например, бумага) для печати документа.
|MediaType|Строка|Используемый по умолчанию носитель (например, бумага) для печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|mediaSize|Строка|Используемый по умолчанию размер носителя. Поддерживает стандартные имена размеров для форматов файлов ISO и ANSI, а также произвольные размеры, поддерживаемые соответствующим принтером.
|pagesPerSheet|Int32|Число страниц документов по умолчанию для печати на каждом листе.
|orientation|принториентатион|Ориентация по умолчанию, используемая при печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|outputBin|Строка|Выходной лоток по умолчанию для размещения завершенных отпечатков. Узнайте о [возможностях](../api/printer-getcapabilities.md) принтера для списка поддерживаемых выходных лотков.|
|фитпдфтопаже|Boolean|Параметр Фитпдфтопаже по умолчанию. Значение true, чтобы разместить каждую страницу документа PDF на физическом листе мультимедиа; false, чтобы разрешить принтеру определять способ макетирования просмотров.|
|multipageLayout|принтмултипажелайаут|Направление по умолчанию для размещения страниц при печати нескольких страниц на листе. Допустимые значения описаны в приведенной ниже таблице.|
|колормоде|принтколормоде|Цветовой режим по умолчанию, используемый при печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|отображения|принткуалити|Качество, используемое по умолчанию при печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|дуплексмоде|принтдуплексмоде|Дуплексная (двухсторонняя) конфигурация, используемая по умолчанию при печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|dpi|Int32|Разрешение по умолчанию в точках на дюйм, используемое при печати задания.|
|scaling|принтскалинг|Указывает, как принтер масштабирует данные документа в соответствии с запрошенным носителем. Допустимые значения описаны в приведенной ниже таблице.|

### <a name="printmultipagelayout-values"></a>значения Принтмултипажелайаут

|Элемент|Значение|Описание|
|:---|:---|:---|
|клокквисефромтоплефт|нуль|Расположите страницы в сетке по часовой стрелке, начиная с верхнего левого угла.|
|каунтерклокквисефромтоплефт|1 |Расположите страницы в сетке против часовой стрелки, начиная с верхнего левого угла.|
|каунтерклокквисефромтопригхт|2 |Расположите страницы в сетке против часовой стрелки, начиная с верхнего правого.|
|клокквисефромтопригхт|4|Расположите страницы в сетке по часовой стрелке, начиная с верхнего правого.|
|каунтерклокквисефромботтомлефт|4 |Расположите страницы в сетке против часовой стрелки, начиная с левой нижней части.|
|клокквисефромботтомлефт|5 |Расположите страницы в сетке по часовой стрелке, начиная с левой нижней части.|
|каунтерклокквисефромботтомригхт|6 |Расположите страницы в сетке против часовой стрелки, начиная с правого нижнего края.|
|клокквисефромботтомригхт|7 |Расположите страницы в сетке по часовой стрелке, начиная с правого нижнего края.|

### <a name="printduplexmode-values"></a>значения Принтдуплексмоде

|Элемент|Значение|Описание|
|:---|:---|:---|
|флипонлонжедже|нуль|Принтер будет печататься на двух сторонах и будет переворачивать документы вдоль длинного края.|
|флипоншортедже|1 |Принтер будет печататься на двух сторонах и будет переражать документы вдоль короткого края.|
|онесидед|2 |На принтере будет напечатана только одна сторона.|

### <a name="printfinishing-values"></a>значения Принтфинишинг

|Элемент|Значение|Описание|
|:---|:---|:---|
|Нет|4|Нет окончаний. В том числе это значение эквивалентно указанию пустой коллекции завершений.|
|сшивание|4 |Скрепка документа с использованием стандартной конфигурации сшивания для принтера.|
|Дырокол|5 |Перфорация документа с использованием стандартной конфигурации перфорации для принтера.|
|возможностью|6 |Примените к документу обложку.|
|базу|7 |Привяжите документ, используя конфигурацию привязки по умолчанию для принтера.|
|саддлеститч|8 |Брошюровка — Стич документ, используя настройку сшивания по умолчанию для принтера.|
|ститчедже|9 |Пограничный — разсшивка документ с использованием стандартной настройки сшивания для принтера.|
|стаплетоплефт|двадцать|Сшивать документ в левом верхнем углу.|
|стаплеботтомлефт|21|Сшивать документ в левом нижнем углу.|
|стаплетопригхт|22|Сшивать документ в правом верхнем углу.|
|стаплеботтомригхт|23|Скрепка документа в правом нижнем углу.|
|ститчлефтедже|открыт|Пограничный — сшивка документа вдоль левого края.|
|ститчтопедже|25|Пограничный — сшивка документа вдоль верхнего края.|
|ститчригхтедже|26|Пограничный — сшивка документа вдоль правого края.|
|ститчботтомедже|27|Пограничный — сшивка документа вдоль нижнего края.|
|стапледуаллефт|8|Сшивать документ дважды вдоль левого края.|
|стапледуалтоп|суммируемых|Сшивать документ дважды вдоль верхнего края.|
|стапледуалригхт|более|Сшивать документ дважды по правому краю.|
|стапледуалботтом|длиной|Сшивать документ дважды вдоль нижнего края.|
|unknownFutureValue|32|Значение Sentinel для перечисления расширяемые. Не следует использовать.|

## <a name="printorientation-values"></a>значения Принториентатион

|Элемент|Значение|Описание|
|:---|:---|:---|
|ориентацию|4|Принтер будет печатать впечатления в "книжной" ориентации.|
|вдоль|4 |Принтер будет печатать впечатления на "альбомной" ориентации.|
|реверселандскапе|5 |Принтер будет печатать впечатления на ориентации "Обратная ориентация".|
|реверсепортраит|6 |Принтер будет печатать впечатления на ориентации "Обратная книжная".|

### <a name="printquality-values"></a>значения Принткуалити

|Элемент|Значение|Описание|
|:---|:---|
|потребление|нуль|Принтер выполнит печать задания, используя низкую (обычно называемую "черновой") качеством.|
|medium|1 |Принтер выполнит печать задания, используя медим (обычно известное как "нормальное").|
|высокоуровневых|2 |Принтер выполнит печать задания, используя высокое (обычное, известное как "наилучшее" или "точное") качество.|
|unknownFutureValue|4|Значение Sentinel для перечисления расширяемые. Не следует использовать.|

### <a name="printcolormode-values"></a>значения Принтколормоде

|Элемент|Значение|Описание|
|:---|:---|:---|
|блаккандвхите|нуль|Черно-белое (используйте только черные материалы с маркерами).|
|черн|1 |Оттенки серого (могут использоваться некоторые материалы цветового маркера).|
|color|2 |Color (Используйте любую комбинацию материалов маркера, чтобы создать впечатление цвета).|
|Авто|4|Разрешите принтеру выбрать используемый цветовой режим.|

### <a name="printscaling-values"></a>значения Принтскалинг

|Элемент|Значение|Описание|
|:---|:---|:---|
|Авто|нуль|Если размер документа превышает запрошенный носитель, а поля не равны нулю, принтер масштабирует **документ, как в принтскалинг.** В противном случае принтер масштабирует документ с помощью **Fill** принтскалинг. Если документ меньше запрошенного медиа-файла, то используется параметр "None" Принтскалинг.|
|shrinkToFit|1 |Если размер документа превышает запрошенный носитель, принтер масштабирует **документ, как в принтскалинг.** В противном случае принтер масштабирует документ, например, как **None** принтскалинг.|
|fill|2 |Принтер масштабирует документ, чтобы заполнить запрошенный размер мультимедиа, сохраняя его пропорции, но при этом могут обрезаться части документа.|
|полностью|4|Принтер масштабирует документ в соответствии с печатной областью запрошенного размера мультимедиа, сохраняя пропорции данных документа без кадрирования документа.|
|Нет|4 |Принтер не масштабирует документ в соответствии с требуемым размером носителя. Если размер документа превышает запрошенный носитель, то принтерные центры и обходят полученные результаты. Если документ меньше запрошенного медиа-файла, принтеры выводят полученные результаты.|
|unknownFutureValue|5 |Значение Sentinel для перечисления расширяемые. Не следует использовать.|
