# <a name="devicecustomattributevaluetype-enum-type"></a>тип перечисления Девицекустоматтрибутевалуетипе

Пространство имен: microsoft.graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Представляет ожидаемый тип для значения скрипта пользовательского атрибута macOS.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|integer|нуль|Указывает, что значение для скрипта настраиваемого атрибута является целым числом.|
|строка|1,1|Указывает, что значение для скрипта настраиваемого атрибута является строкой.|
|dateTime|2|Указывает, что значение для сценария настраиваемого атрибута соответствует стандарту ISO 8601.|
