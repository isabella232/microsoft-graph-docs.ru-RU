# <a name="managementstate-enum-type"></a>тип перечисления Манажементстате

Пространство имен: Microsoft. Graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Состояние управления для устройства в Microsoft Intune.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|которыми|нуль|Устройство находится в управлении|
|ретирепендинг|1 |Команда снятия с учета выполняется на устройстве и в процессе отмены регистрации из управления|
|ретирефаилед|2 |На устройстве не удалось выполнить команду снятия с учета|
|випепендинг|3 |На устройстве и в процессе отмены регистрации в управлении возникает команда очистки.|
|випефаилед|4 |Не удалось выполнить команду очистки на устройстве|
|неисправности|5 |Устройство неработоспособно.|
|делетепендинг|6 |На устройстве выполняется команда Delete |
|ретиреиссуед|7 |Для устройства была выпущена команда снятия с учета|
|випеиссуед|8 |Для устройства была выдана команда очистки.|
|випеканцелед|9 |Команда стирания для этого устройства отменена|
|ретиреканцелед|10 |Команда снятия с учета для этого устройства отменена|
|были|11 |Устройство обнаружено, но не зарегистрировано полностью.|
