# <a name="mobileappsupersedencetype-enum-type"></a>тип перечисления Мобилеаппсуперседенцетипе

Пространство имен: microsoft.graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Указывает тип замены, связанный с отношением между двумя мобильными приложениями.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|исправления|нуль|Указывает, что дочернее приложение должно обновляться внутренней логикой родительского приложения.|
|replace|1 |Указывает, что дочернее приложение должно быть удалено перед установкой родительского приложения.|
