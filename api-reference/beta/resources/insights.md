# <a name="insights-resource-type"></a>Тип ресурса полезные сведения о

> **Важно!** API бета-версии (/beta) в Microsoft Graph проходят тестирование и могут быть изменены. Использование этих API в производственных приложениях не поддерживается.

Полезные сведения о являются связи вычисляется с помощью расширенных аналитических и обучения методики компьютера. К примеру, могут определять документов OneDrive прибора вокруг пользователей.

Полезные сведения о возвращаются следующие интерфейсы API:

- [Тенденций](insights-trending.md) - возвращает документов из OneDrive и сайты SharePoint прибора вокруг пользователя.
- [Используется](insights-used.md) - возвращает документов просматривать и изменять пользователем. Содержит документы, используемые для пользователя в OneDrive для бизнеса, SharePoint, открывается в виде вложений электронной почты, а также как вложения ссылка из источников, например поле, общего банка данных и Google диска.
- [Shared](insights-shared.md) - возвращает документы совместно с пользователем. Документы могут совместно использоваться как вложения электронной почты или OneDrive для бизнеса ссылки в отправленных по электронной почте.

Каждый insight возвращается вместе с `resourceVisualization` и `resourceReference` тип сложных значения (CVT). ResourceVisualization CVT содержит свойства, такие как `title` и `previewImageUrl`. Корпорация Майкрософт использует свойства визуализации для обработки файлов в каждый раз, как углубимся Office.

Содержит документы, используемые для пользователя в OneDrive для бизнеса, SharePoint, открывается в виде вложений электронной почты, а также как вложения ссылка из источников, например поле, общего банка данных и Google диска.  |
