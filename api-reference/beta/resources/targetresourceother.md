# <a name="targetresourceother-resource-type"></a>Тип ресурса targetResourceOther
Указывает сведения о целевой объект, который не могут быть разделены на остальных категорий ресурсов конечного пользователя, приложения, роли, группы, ключ. На основе [targetResource](targetresource.md) ресурсов.

Нет
