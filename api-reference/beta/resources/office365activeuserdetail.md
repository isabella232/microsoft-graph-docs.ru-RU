# <a name="office365activeuserdetail-resource-type"></a>Тип ресурса office365ActiveUserDetail

## <a name="properties"></a>Свойства

| Свойство                          | Тип              | Описание                              |
| :-------------------------------- | :---------------- | ---------------------------------------- |
| Репортрефрешдате                 | Дата              | Самая поздняя дата контента.          |
| userPrincipalName                 | String            | Имя участника-пользователя (UPN). UPN — это имя входа в стиле Интернета для пользователя на основе стандартного стандарта RFC 822. По соглашению он должен сопоставляться с именем электронной почты пользователя. Общий формат — псевдоним @ Domain, где домен должен присутствовать в коллекции проверенных доменов клиента. Это свойство обязательно указывать при создании пользователя. |
| displayName                       | Строка            | Имя пользователя, отображаемое в адресной книге. Обычно это сочетание имени, отчества и фамилии пользователя. Это свойство необходимо указывать при создании пользователя. Его невозможно удалить при обновлении. |
| isDeleted                         | Boolean           | Указывает, был ли этот пользователь удален или обратимо удален. |
| Делетеддате                       | Дата              | Дата, когда произошел сбой операции удаления. Значение по умолчанию — "null", если пользователь не был удален. |
| Хасексчанжелиценсе                | Boolean           | Назначено ли пользователю лицензию на Exchange. |
| Хасонедривелиценсе                | Boolean           | Назначена ли пользователю лицензия на OneDrive. |
| Хасшарепоинтлиценсе              | Boolean           | Назначена ли пользователю лицензия SharePoint. |
| Хасскипефорбусинесслиценсе        | Boolean           | Назначено ли пользователю лицензию Skype для бизнеса. |
| Хасяммерлиценсе                  | Boolean           | Назначена ли пользователю лицензия Yammer. |
| Хастеамслиценсе                   | Boolean           | Назначена ли пользователю лицензия Teams. |
| Ексчанжеластактивитидате          | Дата              | Дата последнего считывания или отправки электронной почты пользователем. |
| Онедривеластактивитидате          | Дата              | Дата, когда пользователь последний раз просматривал или редактировал файлы, общие файлы внутренне или внешне или синхронизированные файлы. |
| Шарепоинтластактивитидате        | Дата              | Дата, когда пользователь последний раз просматривал или редактировал файлы, общие файлы внутренне или внешне, синхронизированные файлы или просмотренные страницы SharePoint. |
| Скипефорбусинессластактивитидате  | Дата              | Дата последнего упорядочения или участие пользователя в конференциях или присоединяемых одноранговых сеансов. |
| Яммерластактивитидате            | Дата              | Дата последнего опубликованного, прочитанного или понравившиего пользователю сообщения. |
| Теамсластактивитидате             | Дата              | Дата последнего отправления пользователем сообщений в каналах группы, отправленных сообщений в сеансах частного чата или участие в собраниях или вызовах. |
| Ексчанжелиценсеассигндате         | Дата              | Последняя дата, когда пользователю назначена лицензия на Exchange. |
| Онедривелиценсеассигндате         | Дата              | Последняя дата, когда пользователю назначена лицензия на OneDrive. |
| Шарепоинтлиценсеассигндате       | Дата              | Последняя дата, когда пользователю назначена лицензия SharePoint. |
| Скипефорбусинесслиценсеассигндате | Дата              | Последняя дата, когда пользователю назначена лицензия Skype для бизнеса. |
| Яммерлиценсеассигндате           | Дата              | Последняя дата, когда пользователю назначена лицензия Yammer. |
| Теамслиценсеассигндате            | Дата              | Последняя дата, когда пользователю назначена лицензия Teams. |
| Ассигнедпродуктс                  | Коллекция строк | Все продукты, назначенные пользователю.  |
