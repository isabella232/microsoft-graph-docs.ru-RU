# <a name="mfadetail-resource-type"></a>Тип ресурса mfaDetail
Указывает сведения о многофакторной проверкой Подлинности для конкретного учетное. Он содержит метод проверки подлинности, используемый для подписи, а также сведения о проверкой подлинности на основе (например: телефон, SMS или голосовая почта) 



## <a name="properties"></a>Свойства
| Свойство     | Тип   |Описание|
|:---------------|:--------|:----------|
|authDetail|Строка|Указывает детализации проверкой подлинности на основе многофакторной проверкой Подлинности для соответствующего действия входа при многофакторной проверкой Подлинности, который требуется — «Да».|
|метод проверки подлинности|Строка|Указывает многофакторной проверкой Подлинности методов проверки подлинности (SMS, телефонную связь, проверки подлинности приложения представлены некоторые значения) для соответствующего действия входа при многофакторной проверкой Подлинности необходимые поля — «Да».|
