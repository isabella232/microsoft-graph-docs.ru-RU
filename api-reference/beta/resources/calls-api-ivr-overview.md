# <a name="ivr-scenarios-in-calls"></a>Сценариев IVR в вызовах

> **Важно!** API бета-версии (/beta) в Microsoft Graph проходят тестирование и могут быть изменены. Использование этих API в производственных приложениях не поддерживается.

Ниже приведены сценарии интерактивного речевого ответа (IVR), которые поддерживают вызова API в Microsoft Graph.

- Воспроизведение звука строки — например, при звонок в очереди сотрудник службы поддержки пользователей.
- Запишите — например, запись звука звонящего абонента, обычно после их Херд строки с параметрами.
- Подпишитесь на тонового сигнала — например, если необходимо знать, какие DTMF звонков вызывающего абонента, обычно после слуха Звуковое приглашение выбрать.
- Например, Отмена обработки мультимедиа - флажок Отменить все операции PlayPrompt или записи, которые находятся в процессе.
