# <a name="iosnotificationpreviewvisibility-enum-type"></a>тип перечисления Иоснотификатионпревиеввисибилити

Пространство имен: microsoft.graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Определяет, отображаются ли эскизы уведомлений на устройстве с iOS. Предварительный просмотр может включать такие сведения, как текст (из сообщений и почта), а также сведения о приглашении (из календаря). При настройке это приведет к переопределению пользовательских параметров предварительного просмотра.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|notConfigured|нуль|Параметры предварительного просмотра уведомлений не будут перезаписаны.|
|алвайсшов|1,1|Всегда показывать предварительные Просмотр уведомлений.|
|хидевхенлоккед|2|Показывать только предварительный просмотр уведомлений, только когда устройство разблокировано.|
|невершов|4|Никогда не показывать предварительные Просмотр уведомлений.|
