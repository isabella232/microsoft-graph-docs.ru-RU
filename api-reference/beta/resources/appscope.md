# <a name="appscope-resource-type"></a>Тип ресурса Аппскопе

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Область назначения роли определяет набор ресурсов, доступ к которым предоставлен участнику. Область приложения — это область, определенная и понятная определенному приложению. Другой тип области — область действия каталога. Области каталогов — это общие области, которые хранятся в каталоге, который понимается несколькими приложениями. 

Он используется как для одного участника, так и для одной однообъектной области, так и для нескольких субъектов.

## <a name="methods"></a>Методы
Нет

## <a name="properties"></a>Свойства

| Свойство | Тип | Описание |
|:-------- |:---- |:----------- |
| id | string | Идентификатор контейнера или ресурса конкретного приложения, представляющие область назначения. Обычно это неизменяемый идентификатор ресурса. Область назначения определяет набор ресурсов, доступ к которым предоставлен участнику. Это свойство является обязательным. |
| type | String | Описывает тип ресурса для конкретного приложения, представленного областью приложения. Позволяет пользовательскому интерфейсу передать пользователю тип ресурса приложения, представленного областью приложения. Это свойство доступно только для чтения. |
| displayName | string | Предоставляет отображаемое имя ресурса для конкретного приложения, представленного областью приложения. Используется для отображения в целях отображения, так как Аппскопеид часто является неизменяемым идентификатором, не предназначенным для человека. Это свойство доступно только для чтения. |

Нет
