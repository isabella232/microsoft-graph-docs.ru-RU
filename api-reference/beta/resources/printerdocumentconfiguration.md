# <a name="printerdocumentconfiguration-resource-type"></a>Тип ресурса Принтердокументконфигуратион

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Группа параметров, которые принтер должен использовать для печати документа.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|пажеранжес|Коллекция [принтпажеранже](printpagerange.md)|Диапазоны страниц для печати. Только для чтения.|
|принткуалити|принткуалити|Качество печати, используемое при печати задания. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|принтресолутиониндпи|Int32|Решение, используемое при печати задания, выраженное в точках на дюйм (DPI). Только для чтения.|
|фиддиректион|принтерфиддиректион|Направление, используемое при выдачах мультимедиа на принтер. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|orientation|принториентатион|Параметр Orientation, который должен использоваться принтером при печати задания. Допустимые значения описаны в приведенной ниже таблице.|
|дуплексконфигуратион|принтдуплексконфигуратион|Дуплексная конфигурация, которая должна использоваться принтером при печати задания. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|скопирует|Int32|Количество копий, которое необходимо напечатать. Только для чтения.|
|колорконфигуратион|принтколорконфигуратион|Цветовая конфигурация, которую должен использовать принтер для печати задания. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|

### <a name="printduplexconfiguration-values"></a>значения Принтдуплексконфигуратион

|Элемент|Значение|Описание|
|:---|:---|:---|
|твосидедлонжедже|нуль|Принтер будет печататься на двух сторонах и будет переворачивать документы вдоль длинного края.|
|твосидедшортедже|1,1|Принтер будет печататься на двух сторонах и будет переражать документы вдоль короткого края.|
|онесидед|2|На принтере будет напечатана только одна сторона.|

### <a name="printquality-values"></a>значения Принткуалити

|Member|Описание|
|:---|:---|
|потребление|Принтер выполнит печать задания, используя низкую (обычно называемую "черновой") качеством.|
|medium|Принтер выполнит печать задания, используя медим (обычно известное как "нормальное").|
|высокоуровневых|Принтер выполнит печать задания, используя высокое (обычное, известное как "наилучшее" или "точное") качество.|

## <a name="printerfeeddirection-values"></a>значения Принтерфиддиректион

|Member|Описание|
|:---|:---|
|лонжеджефирст|Принтер будет использовать листы из активного лотка на "альбомной" ориентации, с длинным краем листа.|
|шортеджефирст|Принтер будет использовать листы из активного лотка на странице "Книжная" (ориентация), при этом сначала используется короткий край листа.|

## <a name="printorientation-values"></a>значения Принториентатион

|Элемент|Значение|Описание|
|:---|:---|:---|
|ориентацию|4|Принтер будет печатать впечатления в "книжной" ориентации.|
|вдоль|4 |Принтер будет печатать впечатления на "альбомной" ориентации.|
|реверселандскапе|5 |Принтер будет печатать впечатления на ориентации "Обратная ориентация".|
|реверсепортраит|6 |Принтер будет печатать впечатления на ориентации "Обратная книжная".|

### <a name="printcolorconfiguration-values"></a>значения Принтколорконфигуратион

|Элемент|Значение|Описание|
|:---|:---|:---|
|блаккандвхите|нуль|Черно-белое (используйте только черные материалы с маркерами).|
|черн|1,1|Оттенки серого (могут использоваться некоторые материалы цветового маркера).|
|color|2|Color (Используйте любую комбинацию материалов маркера, чтобы создать впечатление цвета).|
|Авто|4|Разрешите принтеру выбрать используемый цветовой режим.|
