# <a name="printerdocumentconfiguration-resource-type"></a>Тип ресурса Принтердокументконфигуратион

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Группа параметров, которые принтер должен использовать для печати документа.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|пажеранжес|Коллекция [интежерранже](integerrange.md)|Диапазоны страниц для печати. Только для чтения.|
|отображения|принткуалити|Качество печати, используемое при печати задания. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|600|Int32|Решение, используемое при печати задания, выраженное в точках на дюйм (DPI). Только для чтения.|
|фиддиректион|принтерфиддиректион|Направление, используемое при выдачах мультимедиа на принтер. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|orientation|принториентатион|Параметр Orientation, который должен использоваться принтером при печати задания. Допустимые значения описаны в приведенной ниже таблице.|
|дуплексмоде|принтдуплексмоде|Дуплексный режим, который должен использоваться принтером при печати задания. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|скопирует|Int32|Количество копий, которое необходимо напечатать. Только для чтения.|
|колормоде|принтколормоде|Цветовой режим, который должен использоваться принтером для печати задания. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|инпутбин|String|Входной лоток (табло), используемый при печати. Узнайте о [возможностях](../api/printer-getcapabilities.md) принтера для списка поддерживаемых входных лотков.|
|аутпутбин|String|Выходной лоток для размещения завершенных отпечатков. Узнайте о [возможностях](../api/printer-getcapabilities.md) принтера для списка поддерживаемых выходных лотков.|
|медиасизе|String|Сизето мультимедиа, используемый при печати. Поддерживает стандартные имена размеров для форматов файлов ISO и ANSI, а также произвольные размеры, поддерживаемые соответствующим принтером.|
|поля|[принтмаргин](printmargin.md)|Параметры полей, используемые при печати.|
|mediaType|String|Используемый по умолчанию носитель (например, бумага) для печати документа. Допустимые значения описаны в приведенной ниже таблице.|
|завершения|Коллекция Принтфинишинг|Завершение процессов, которые необходимо использовать при печати.|
|пажеспершит|Int32|Число страниц документа, которые должны печататься на каждом листе.
|мултипажелайаут|принтмултипажелайаут|Направление размещения страниц при печати нескольких страниц на листе. Допустимые значения описаны в приведенной ниже таблице.|
|Сортировка по копиям|Boolean|Указывает, должен ли принтер разбирать страницы Вехен печати нескольких копий многостраничного документа.|
|масштабирование|принтскалинг|Указывает, как принтер должен масштабировать данные документа в соответствии с запрошенным носителем. Допустимые значения описаны в приведенной ниже таблице.|

### <a name="printquality-values"></a>значения Принткуалити

|Элемент|Значение|Описание|
|:---|:---|
|потребление|нуль|Принтер выполнит печать задания, используя низкую (обычно называемую "черновой") качеством.|
|medium|1 |Принтер выполнит печать задания, используя медим (обычно известное как "нормальное").|
|высокоуровневых|2 |Принтер выполнит печать задания, используя высокое (обычное, известное как "наилучшее" или "точное") качество.|
|unknownFutureValue|3 |Значение Sentinel для перечисления расширяемые. Не следует использовать.|

## <a name="printerfeeddirection-values"></a>значения Принтерфиддиректион

|Элемент|Значение|Описание|
|:---|:---|
|лонжеджефирст|нуль|Принтер будет использовать листы из активного лотка на "альбомной" ориентации, с длинным краем листа.|
|шортеджефирст|1 |Принтер будет использовать листы из активного лотка на странице "Книжная" (ориентация), при этом сначала используется короткий край листа.|
|unknownFutureValue|2 |Значение Sentinel для перечисления расширяемые. Не следует использовать.|

## <a name="printorientation-values"></a>значения Принториентатион

|Элемент|Значение|Описание|
|:---|:---|:---|
|ориентацию|3 |Принтер будет печатать впечатления в "книжной" ориентации.|
|вдоль|4 |Принтер будет печатать впечатления на "альбомной" ориентации.|
|реверселандскапе|5 |Принтер будет печатать впечатления на ориентации "Обратная ориентация".|
|реверсепортраит|6 |Принтер будет печатать впечатления на ориентации "Обратная книжная".|

### <a name="printduplexmode-values"></a>значения Принтдуплексмоде

|Элемент|Значение|Описание|
|:---|:---|:---|
|флипонлонжедже|нуль|Принтер будет печататься на двух сторонах и будет переворачивать документы вдоль длинного края.|
|флипоншортедже|1 |Принтер будет печататься на двух сторонах и будет переражать документы вдоль короткого края.|
|онесидед|2 |На принтере будет напечатана только одна сторона.|

### <a name="printcolormode-values"></a>значения Принтколормоде

|Элемент|Значение|Описание|
|:---|:---|:---|
|блаккандвхите|нуль|Черно-белое (используйте только черные материалы с маркерами).|
|черн|1 |Оттенки серого (могут использоваться некоторые материалы цветового маркера).|
|color|2 |Color (Используйте любую комбинацию материалов маркера, чтобы создать впечатление цвета).|
|Авто|3 |Разрешите принтеру выбрать используемый цветовой режим.|

### <a name="printscaling-values"></a>значения Принтскалинг

|Элемент|Значение|Описание|
|:---|:---|:---|
|Авто|нуль|Если размер документа превышает запрошенный носитель, а поля не равны нулю, принтер масштабирует **документ, как в принтскалинг.** В противном случае принтер масштабирует документ с помощью **Fill** принтскалинг. Если документ меньше запрошенного медиа-файла, то используется параметр "None" Принтскалинг.|
|shrinkToFit|1 |Если размер документа превышает запрошенный носитель, принтер масштабирует **документ, как в принтскалинг.** В противном случае принтер масштабирует документ, например, как **None** принтскалинг.|
|fill|2 |Принтер масштабирует документ, чтобы заполнить запрошенный размер мультимедиа, сохраняя его пропорции, но при этом могут обрезаться части документа.|
|полностью|3 |Принтер масштабирует документ в соответствии с печатной областью запрошенного размера мультимедиа, сохраняя пропорции данных документа без кадрирования документа.|
|Нет|4 |Принтер не масштабирует документ в соответствии с требуемым размером носителя. Если размер документа превышает запрошенный носитель, то принтерные центры и обходят полученные результаты. Если документ меньше запрошенного медиа-файла, принтеры выводят полученные результаты.|
|unknownFutureValue|5 |Значение Sentinel для перечисления расширяемые. Не следует использовать.|
