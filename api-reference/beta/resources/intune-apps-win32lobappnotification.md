# <a name="win32lobappnotification-enum-type"></a>Тип перечисления win32LobAppNotification

> **Важные:** Интерфейсы API в разделе версии /beta в Microsoft Graph могут быть изменены. Использование этих API в производственных приложениях не поддерживается.

> **Примечание:** Microsoft Graph API для Intune требуется [Активная лицензия Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Содержит значение для уведомления о состоянии.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|showAll|0|Отображение всех уведомлений.|
|showReboot|1|Показывать только перезапустите уведомлений и отменить другие виды уведомления.|
|hideAll|2|Скройте все уведомления.|
