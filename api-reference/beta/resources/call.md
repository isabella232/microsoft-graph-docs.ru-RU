# <a name="call-resource-type"></a><span data-ttu-id="ffb31-103">Тип ресурса call</span><span class="sxs-lookup"><span data-stu-id="ffb31-103">call resource type</span></span>

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

<span data-ttu-id="ffb31-104">Ресурс **call** создается при наличии входящего вызова приложения или при создании приложением нового исходящего вызова с помощью запроса `POST` в `app/calls`.</span><span class="sxs-lookup"><span data-stu-id="ffb31-104">The **call** resource is created when there is an incoming call for the application or the application creates a new outgoing call via a `POST` on `app/calls`.</span></span>
