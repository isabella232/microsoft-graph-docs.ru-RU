# <a name="windowsupdatestatus-enum-type"></a>тип перечисления Виндовсупдатестатус

Пространство имен: Microsoft. Graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Состояния устройств конфигурации центра обновления Windows для бизнеса

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|уптодате|нуль|Нет ожидающих обновлений, ожидающих обновлений для перезагрузки и без сбоев обновлений.|
|пендингинсталлатион|1 |Существуют обновления, ожидающие установки, которые включают Неутвержденные обновления. Нет обновлений, ожидающих перезагрузки, неудачных обновлений.|
|пендингребут|2 |Существуют обновления, требующие перезагрузки. Обновления не выполнены.|
|сбоев|3 |Не удалось установить обновления на устройстве.|
