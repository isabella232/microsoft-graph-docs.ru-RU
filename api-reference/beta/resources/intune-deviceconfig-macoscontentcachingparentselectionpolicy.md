# <a name="macoscontentcachingparentselectionpolicy-enum-type"></a>тип перечисления Макосконтенткачингпарентселектионполици

Пространство имен: microsoft.graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Определяет, как кэширование содержимого выбирает родительский кэш.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|notConfigured|нуль|По умолчанию используется стратегия циклического перебора.|
|roundRobin|1 |Поворачивайте родительские элементы по порядку. Используйте эту политику для балансировки нагрузки.|
|фирставаилабле|2|Всегда используйте первый доступный родительский элемент в списке родителей. Эта политика используется для назначения постоянных основных, вторичных и последующих родительских элементов.|
|урлпасхаш|4|Хэширует часть пути в запрашиваемом URL-адресе, чтобы для одного URL-адреса всегда использовался один и тот же родительский элемент. Это полезно для максимизации размера Объединенных кэшей родителей.|
|произвольную|4 |Выберите родительский элемент случайным образом. Используйте эту политику для балансировки нагрузки.|
|стиккяваилабле|5 |Используйте первый доступный родительский элемент, который доступен в списке родителей, пока он не станет недоступен, а затем перейдите к следующему. Эта политика используется для назначения плавающего основного, дополнительного и последующего родительских элементов.|
