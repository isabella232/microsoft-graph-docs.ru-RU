# <a name="windowsdefenderapplicationcontrolsupplementalpolicystatuses-enum-type"></a>тип перечисления Виндовсдефендераппликатионконтролсупплементалполицистатусес

Пространство имен: Microsoft. Graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Перечисление значений различных состояний развертывания дополнительной политики Виндовсдефендераппликатионконтрол.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|нуль|Состояние дополнительной политики Виндовсдефендераппликатионконтрол неизвестно.|
|success|1 |Применяется дополнительная политика Виндовсдефендераппликатионконтрол.|
|токенеррор|2 |Дополнительная политика Виндовсдефендераппликатионконтрол имеет структуру, но существует ошибка, связанная с авторизацией маркера.|
|нотаусоризедбитокен|3 |Маркер не авторизует эту дополнительную политику Виндовсдефендераппликатионконтрол.|
|полицинотфаунд|4 |Дополнительная политика Виндовсдефендераппликатионконтрол не найдена.|
