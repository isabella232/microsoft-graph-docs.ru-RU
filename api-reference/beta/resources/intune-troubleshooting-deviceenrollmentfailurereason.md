Пространство имен: Microsoft. Graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Категории отказов верхнего уровня для регистрации.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|нуль|Значение по умолчанию, причина сбоя неизвестна.|
|проверки подлинности|1 |Ошибка проверки подлинности|
|проверке|2 |Вызов прошел проверку подлинности, но не авторизован для регистрации.|
|аккаунтвалидатион|3 |Не удалось проверить учетную запись для регистрации. (Учетная запись заблокирована, регистрация не включена)|
|усервалидатион|4 |Не удалось проверить пользователя. (Пользователь не существует, отсутствует лицензия)|

|необслуживание|6 |Учетная запись находится в состоянии обслуживания.|
|бадрекуест|7 |Клиент отправил запрос, который не распознаются или не поддерживается службой.|
|феатуренотсуппортед|8 |Функции, используемые этой регистрацией, не поддерживаются для этой учетной записи.|
|енроллментрестриктионсенфорцед|9 |Ограничения на регистрацию, настроенные администратором, заблокировали эту регистрацию.|

|усерабандонмент|11 |Регистрация отменена ендусер. (Ендусер запустил входящую миграцию, но не смог выполнить ее своевременно)|
