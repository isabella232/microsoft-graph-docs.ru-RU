# <a name="chatmessagepolicyviolation-resource-type"></a>Тип ресурса Чатмессажеполицивиолатион

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет нарушение политики в сообщении чата. Нарушения политики обычно задаются с помощью приложения предотвращения потери данных (DLP). Приложения DLP отслеживают сеансы обмена сообщениями, которые содержат данные, которые не предполагается отправлять пользователям.

## <a name="properties"></a>Свойства

| Свойство   | Тип |Описание|
|:---------------|:--------|:----------|
|длпактион|**чатмессажеполицивиолатиондлпактионтипе**|Действие, выполняемое поставщиком DLP для сообщения с конфиденциальным содержимым. Поддерживаемые значения: <li>Нет</li><li>NotifySender — уведомить отправителя о нарушении, но позволить читателям читать сообщение.</li><li>Блоккакцесс запрещает читателям читать сообщение.</li><li>Блоккакцессекстернал — блокировать чтение сообщения пользователями за пределами Организации, в то же время позволяя пользователям в Организации читать сообщение.</li>|
|жустификатионтекст|string|Текст обоснования, предоставленный отправителем сообщения при переопределении нарушения политики.|
|полицитип|[чатмессажеполицивиолатионполицитип](chatmessagepolicyviolationpolicytip.md)|Сведения, которые должны отображаться у отправителя сообщения о причинах пометки сообщения как нарушения. |
|усерактион|**чатмессажеполицивиолатионусерактионтипе**|Указывает действие, выполняемое пользователем для сообщения, заблокированного поставщиком DLP. Поддерживаемые значения: <li>Нет</li><li>Override</li><li>репортфалсепоситиве</li>Когда поставщик DLP обновляет сообщение для блокирования конфиденциального содержимого, Усерактион не является обязательным.|
|вердиктдетаилс|**чатмессажеполицивиолатионвердиктдетаилстипе**|Указывает, какие действия может предпринять отправитель в ответ на нарушение политики. Поддерживаемые значения: <li>Нет</li><li>Алловфалсепоситивеоверриде — позволяет отправителю объявить сообщение об ошибке в приложении DLP и его правилах и позволить читателям снова видеть сообщение, если оно было скрыто в Длпактион.</li><li>Аллововерридевисаутжустификатион — позволяет отправителю оверрииде нарушение защиты от потери данных и позволить читателям снова видеть сообщение, если оно было скрыто в Длпактион, без необходимости предоставлять объяснение для этого. </li><li>Аллововерридевисжустификатион — позволяет отправителю оверрииде нарушение защиты от потери данных и позволить читателям снова видеть сообщение, если оно было скрыто в Длпактион, после предоставления объяснения для этого.</li>Аллововерридевисаутжустификатион и Аллововерридевисжустификатион являются взаимоисключающими.|
