# <a name="managementconditionexpression-resource-type"></a>Тип ресурса Манажементкондитионекспрессион

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Выражение условия управления — это выражение, которое при оценке возвращает логическое значение, которое указывает, что оператор условия управления активирован/деактивирован. Выражение условия управления может состоять из комбинации переменных Expression и операторов выражений с логическими значениями.
