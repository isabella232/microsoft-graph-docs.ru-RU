# <a name="devicemanagementconfigurationgroupsettinginstance-resource-type"></a>Тип ресурса Девицеманажементконфигуратионграупсеттингинстанце

Пространство имен: microsoft.graph

> **Важно!** API Microsoft Graph в версии/Beta могут изменяться; рабочее использование не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Экземпляр объекта GroupSetting


Наследуется от [девицеманажементконфигуратионсеттингинстанце](../resources/intune-deviceconfigv2-devicemanagementconfigurationsettinginstance.md)

## <a name="properties"></a>Свойства
|Свойство|Тип|Описание|
|:---|:---|:---|
|сеттингдефинитионид|String|Параметр ID определения, наследуемый от [девицеманажементконфигуратионсеттингинстанце](../resources/intune-deviceconfigv2-devicemanagementconfigurationsettinginstance.md)|
|граупсеттингвалуе|[девицеманажементконфигуратионграупсеттингвалуе](../resources/intune-deviceconfigv2-devicemanagementconfigurationgroupsettingvalue.md)|Значение GroupSetting|

## <a name="relationships"></a>Связи
Нет
