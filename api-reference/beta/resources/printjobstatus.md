# <a name="printjobstatus-resource-type"></a>Тип ресурса Принтжобстатус

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет текущее состояние задания печати.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|процессингстате|принтжобпроцессингстате|Текущее состояние обработки задания печати. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|процессингстатедескриптион|Строка|Удобное для человека описание текущего состояния обработки задания печати. Только для чтения.|
|аккуиредбипринтер|Boolean|Значение true, если задание было подтверждено принтером; в противном случае — false. Только для чтения.|

### <a name="printjobprocessingstate-values"></a>значения Принтжобпроцессингстате

|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|нуль|Состояние обработки, сообщаемое принтером, не распознано.|
|закончен|1 |Задание печати ожидает обработки принтером.|
|пендингхелд|2 |Задание еще не является кандидатом на обработку. Убедитесь в отсутствии ошибок или ограничений ресурсов, препятствующих запуску задания.|
|двухпроцессорной|4|Задание печати в настоящее время обрабатывается принтером.|
|приостановлено|4 |Задание печати приостановлено пользователем.|
|отключен|5 |Задание печати остановлено, так как перед продолжением выполнения задания необходимо устранить проблемы с принтером. Дополнительные сведения можно найти в ресурсе "состояние принтера".|
|готовы|6 |Задание печати успешно завершено, дальнейшая обработка выполняться не будет.|
|закрыт|7 |Задание печати отменено пользователем, дальнейшая обработка не будет выполнена.|
|прерван|8 |Задание печати было прервано пользователем или принтером, и дальнейшая обработка не будет выполнена.|
