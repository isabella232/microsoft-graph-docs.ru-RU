# <a name="printjobstatus-resource-type"></a>Тип ресурса Принтжобстатус

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет текущее состояние задания печати.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|state|принтжобпроцессингстате|Текущее состояние обработки задания печати. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|details|Коллекция Принтжобпроцессингдетаил|Дополнительные сведения о состоянии задания печати. Допустимые значения описаны в приведенной ниже таблице. Только для чтения.|
|description|Строка|Удобное для человека описание текущего состояния обработки задания печати. Только для чтения.|
|исаккуиредбипринтер|Логический|Значение true, если задание было подтверждено принтером; в противном случае — false. Только для чтения.|

### <a name="printjobprocessingstate-values"></a>значения Принтжобпроцессингстате

|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|нуль|Состояние обработки, сообщаемое принтером, не распознано.|
|закончен|1,1|Задание печати ожидает обработки принтером.|
|двухпроцессорной|2|Задание печати в настоящее время обрабатывается принтером.|
|приостановлено|4|Задание печати приостановлено.|
|отключен|4 |Задание печати остановлено, так как перед продолжением выполнения задания необходимо устранить проблемы с принтером. Дополнительные сведения можно найти в ресурсе "состояние принтера".|
|готовы|5 |Задание печати успешно завершено, дальнейшая обработка выполняться не будет.|
|закрыт|6 |Задание печати отменено пользователем, дальнейшая обработка не будет выполнена.|
|прерван|7 |Задание печати было прервано пользователем или принтером, и дальнейшая обработка не будет выполнена.|

### <a name="printjobprocessingdetail-values"></a>значения Принтжобпроцессингдетаил

|Элемент|Значение|Описание|
|:---|:---|:---|
|уплоадпендинг|нуль|Полезные данные документа не были отправлены.|
|Преобразование|1,1|Выполняется преобразование полезных данных документа.|
|комплетедсукцессфулли|2|Задание успешно завершено.|
|комплетедвисварнингс|4|Задание было завершено с предупреждениями.|
|комплетедвисеррорс|4 |Задание выполнено с ошибками.|
|релеасеваит|5 |Ожидается выпуск задания.|
|интерпретации|6 |Задание находится в состоянии "обработка", но точнее, интерпретация полезных данных документа.|
