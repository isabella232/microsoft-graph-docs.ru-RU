# <a name="table-resource-type"></a>Тип ресурса Table

Представляет таблицу Excel.


## <a name="methods"></a>Методы

| Метод           | Возвращаемый тип    |Описание|
|:---------------|:--------|:----------|
|[Получение таблицы](../api/table_get.md) | [Таблица книги](table.md) |Чтение свойств и связей объекта таблицы.|
|[Создание объекта TableColumn](../api/table_post_columns.md) |[Колонка в таблице книги](tablecolumn.md)| Создание объекта TableColumn путем добавления в коллекцию столбцов.|
|[Список столбцов](../api/table_list_columns.md) |Коллекция[Столбец в таблице книги](tablecolumn.md)| Получение коллекции объектов TableColumn.|
|[Создание объекта TableRow](../api/table_post_rows.md) |[Ряд в таблице книги](tablerow.md)| Создание объекта TableRow путем добавления в коллекцию строк.|
|[Список строк](../api/table_list_rows.md) |Коллекция[Строка в таблице книги](tablerow.md)| Получение коллекции объектов TableRow.|
|[Обновление](../api/table_update.md) | [Таблица книги](table.md)   |Обновить таблицу |
|[Диапазон корпуса данных](../api/table_databodyrange.md)|[Диапазон](range.md)|Получает объект диапазона, связанный с основными данными таблицы.|
|[Диапазон строки заголовка](../api/table_headerrowrange.md)|[Диапазон](range.md)|Получает объект диапазона, связанный со строкой заголовков таблицы.|
|[Диапазон](../api/table_range.md)|[Диапазон](range.md)|Получает объект диапазона, связанный со всей таблицей.|
|[Полный диапазон строк](../api/table_totalrowrange.md)|[Диапазон](range.md)|Получает объект диапазона, связанный со строкой итогов таблицы.|
|[Очистка фильтров](../api/table_clearfilters.md)|Нет|Удаляет все фильтры, примененные к таблице.|
|[Переход в оранжевый](../api/table_converttorange.md)|[Диапазон](range.md)|Преобразовывает таблицу в обычный диапазон ячеек. Все данные сохраняются.|
|[Удалить](../api/table_delete.md)|Нет|Удаляет таблицу.|
|[Применить фильтры заново](../api/table_reapplyfilters.md)|Нет|Повторно применяет все текущие фильтры к таблице.|
|[Список](../api/table_list.md) | Коллекция[Таблица книги](table.md) |Получение коллекции объектов table. |
|[Добавить](../api/tablecollection_add.md)|[Таблица книги](table.md)|Создание таблицы. Исходный адрес диапазона определяет лист, на который будет добавлена таблица. Если не удается добавить таблицу (например, если адрес недействителен или одна таблица будет перекрываться другой), выводится сообщение об ошибке.|

## <a name="properties"></a>Свойства
| Свойство     | Тип   |Описание|
|:---------------|:--------|:----------|
|идентификатор|строка|Возвращает значение, однозначно идентифицирующее таблицу в данной книге. Значение идентификатора остается прежним, даже если переименовать таблицу. Это свойство должно интерпретироваться как непрозрачное строковое значение и не должно преобразовываться в любой другой тип. Только для чтения.|
|имя|строка|Имя таблицы.|
|показать заголовки|логический|Указывает, отображается ли строка заголовков. Можно задать это значение, чтобы отобразить или скрыть строку заголовков.|
|показать итог|логический|Указывает, отображается ли строка итогов. Можно задать это значение, чтобы отобразить или скрыть строку итогов.|
|стиль|строка|Постоянное значение, представляющее стиль таблицы. Возможные значения: от TableStyleLight1 до TableStyleLight21, от TableStyleMedium1 до TableStyleMedium28, от TableStyleStyleDark1 до TableStyleStyleDark11. Настраиваемый пользователем стиль, имеющийся в книге, также может быть определён.|
|выделить цветом первый столбец|Логический|Указывает, содержит ли первый столбец специальное форматирование.   |
|выделить цветом последний столбец|Логический|Указывает, содержит ли последний столбец специальное форматирование. |
|показать связанные столбцы|Логический|Указывает, чередуется ли форматирование четных и нечетных столбцов для более удобного просмотра таблицы.   |
|показать связанные ряды|Логический|Указывает, чередуется ли форматирование четных и нечетных строк для более удобного просмотра таблицы.    |
|показать кнопку фильтра|Логический|Указывает, видны ли кнопки фильтрации в верхней части заголовков столбцов. Это свойство можно использовать, только если таблица содержит строку заголовков.   |
|идентификатор прежней версии|Строка|Устаревший идентификатор, используемый в старых клиентах Excel. Значение идентификатора остается прежним, даже если переименовать таблицу. Это свойство должно интерпретироваться как непрозрачное строковое значение и не должно преобразовываться в любой другой тип. Только для чтения.   |

## <a name="relationships"></a>Связи
| Связь | Тип   |Описание|
|:---------------|:--------|:----------|
|столбцы|Коллекция[Столбец в таблице книги](tablecolumn.md)|Представляет коллекцию всех столбцов в таблице. Только для чтения.|
|строки|Коллекция[Строка в таблице книги](tablerow.md)|Представляет коллекцию всех строк в таблице. Только для чтения.|
|сортировать|[Сортировать таблицу книги](tablesort.md)|Представляет сортировку для таблицы. Только для чтения.|
|лист|[Лист книги](worksheet.md)|Лист, содержащий текущую таблицу. Только для чтения.|
