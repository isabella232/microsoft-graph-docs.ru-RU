# <a name="devicemanagementexchangeaccessstate-enum-type"></a>Тип перечисления deviceManagementExchangeAccessState

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Состояние доступа к Exchange для устройств.
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|none|0|Состояние доступа к Exchange не определено|
|unknown|1|Состояние доступа устройства к Exchange неизвестно|
|allowed|2|У устройства есть доступ к Exchange|
|blocked|3|Устройство заблокировано в Exchange|
|quarantined|4|Устройство находится в карантине Exchange|
