# <a name="ratingaustraliatelevisiontype-enum-type"></a>Тип перечисления ratingAustraliaTelevisionType

Метки возрастных категорий телевизионного контента в Австралии
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|allAllowed|0|Значение по умолчанию, допуск всего контента телепередач|
|allBlocked|1|Запрет любого контента телепередач|
|preschoolers|2|Категория P — рассчитано на детей дошкольного возраста|
|children|3|Категория C — рассчитано на детей младше 14 лет|
|general|4|Категория G — подходит для любого возраста|
|parentalGuidance|5|Категория PG — рекомендуется для юных зрителей|
|mature|6|Категория M — рекомендуется для зрителей старше 15 лет|
|agesAbove15|7|Категория MA15+ — не подходит для зрителей младше 15 лет|
|agesAbove15AdultViolence|8|Категория AV15+ — не подходит для зрителей младше 15 лет — изображение жестокости|
