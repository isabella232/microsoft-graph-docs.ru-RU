# <a name="androidworkprofilerequiredpasswordtype-enum-type"></a>Тип перечисления androidWorkProfileRequiredPasswordType

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Тип пароля, запрашиваемый рабочим профилем Android.
## <a name="members"></a>Члены
|Член|Значение|Описание|
|:---|:---|:---|
|deviceDefault|0|Значение устройства по умолчанию, без назначения.|
|lowSecurityBiometric|1|Требуется базовый пароль на основе биометрии низкого уровня безопасности.|
|required|2|Обязательный.|
|atLeastNumeric|3|Требуется по крайней мере цифровой пароль.|
|numericComplex|4|Требуется сложный цифровой пароль.|
|atLeastAlphabetic|5|Требуется по крайней мере буквенный пароль.|
|atLeastAlphanumeric|6|Требуется по крайней мере буквенно-цифровой пароль.|
|alphanumericWithSymbols|7|Требуется пароль c цифрами, буквами и символами.|
