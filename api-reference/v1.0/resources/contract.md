# <a name="contract-resource-type"></a>Тип ресурса контракта
Представляет существующую связь, с которой клиент-партнер имеет клиент.

> **Важно!** Существует только в партнерских клиентах. Партнерские клиенты — это клиенты Azure AD, которые принадлежат партнерам Майкрософт, которые входят в состав [поставщика облачных решений корпорации Майкрософт](https://partnercenter.microsoft.com/en-us/partner/programs), Office 365 для синдикации или партнерских программ Microsoft Advisor.

## <a name="methods"></a>Методы

| Метод   | Возвращаемый тип | Описание |
|:---------------|:--------|:----------|
|[Получение контрактов](../api/contract-get.md) | Суммы |Считывание свойств определенного объекта Contract. |
|[Список контрактов](../api/contract-list.md) | Коллекция контрактов | Список контрактов в клиенте партнера. |

## <a name="properties"></a>Свойства
| Свойство   | Тип | Описание |
|:---------------|:--------|:----------|
|Контракттипе|String|Тип контракта.<br><br>Возможные значения:<br> *Синдикатионпартнер* -партнер, который исключительно перепродает и управляет O365 и Intune для этого клиента. Они перепродают и поддерживают своих клиентов.<br> *Бреадспартнер* -Partner предоставляет возможность администрирования для этого клиента. Тем не менее, партнеру не разрешено перепродавать клиенту.<br>*Реселлерпартнер* -партнер, аналогичный партнеру синдикации, за исключением того, что партнер не имеет монопольного доступа к клиенту. В случае синдикации клиент не может покупать дополнительные подписки от Майкрософт или других партнеров.|
|customerId|GUID|Уникальный идентификатор клиента клиента, на который ссылается это партнерство. Соответствует свойству ID ресурса организации клиента клиента. |
|Дефаултдомаиннаме|String|Копия доменного имени клиента, используемого по умолчанию. Копия создается при установке связи с клиентом. Он не обновляется автоматически, если изменяется доменное имя клиента по умолчанию.|
|displayName|Строка|Копия отображаемого имени клиента. Копия создается при установке связи с клиентом. Он не обновляется автоматически при изменении отображаемого имени клиента клиента.|
|id|String| Уникальный идентификатор партнерства. Ключ, только для чтения |
