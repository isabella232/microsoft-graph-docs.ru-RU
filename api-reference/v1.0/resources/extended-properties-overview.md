## <a name="use-extended-properties-or-open-extensions"></a><span data-ttu-id="72a1d-116">Что использовать — расширенные свойства или открытые расширения</span><span class="sxs-lookup"><span data-stu-id="72a1d-116">Use extended properties or open extensions?</span></span>
