## <a name="use-extended-properties-or-open-extensions"></a><span data-ttu-id="e40e9-114">Что использовать — расширенные свойства или открытые расширения</span><span class="sxs-lookup"><span data-stu-id="e40e9-114">Use extended properties or open extensions?</span></span>
