# <a name="compliancestate-enum-type"></a>тип перечисления complianceState

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Состояние соответствия требованиям.
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|0|Неизвестно.|
|compliant|1|Соответствует требованиям.|
|nonCompliant|2|Устройство не соответствует требованиям и заблокировано корпоративными ресурсами.|
|conflict|3|Конфликт с другими правилами.|
|error|4|Ошибка.|
|inGracePeriod|254|Устройство не соответствует требованиям, но всё равно имеет доступ к корпоративным ресурсам|
|configManager|255|Управляется диспетчером конфигурации|
