# <a name="recurrencerange-resource-type"></a>Тип ресурса recurrenceRange

Описывает диапазон дат, согласно которому повторяется [событие](event.md). 

Вы можете задать диапазон дат для повторяющегося события одним из 3 способов (в зависимости от сценария). Необходимо всегда указывать значение **startDate** для диапазона дат, но вы можете, к примеру, задать повторяющееся событие, которое перестает повторяться в определенный день либо повторяется бесконечно или только 5 раз. Обратите внимание, что фактические экземпляры события в диапазоне дат всегда соответствуют расписанию повторения, заданному для повторяющегося события. Повторяющееся событие всегда определяется соответствующими ресурсами [recurrencePattern](recurrencepattern.md) (частотой повторения события) и **recurrenceRange** (продолжительностью повторения).

С помощью свойства **type** можно указывать различные типы для **recurrenceRange**. Обратите внимание на обязательные свойства для каждого типа, описанные в приведенной ниже таблице.

| Тип диапазона повторения | Значение свойства type | Описание | Пример | Обязательные свойства |
|:---------------|:--------|:--------|:--------|:----------|
|Диапазон с датой окончания | `endDate` | Событие повторяется во все дни, входящие в соответствующее расписание повторения, начиная с даты **startDate** и заканчивая датой **endDate**. | Повторение события с 1 июня 2017 г. до 15 июня 2017 г. | **type**, **startDate**, **endDate** | 
|Диапазон без даты окончания | `noEnd` | Событие повторяется во все дни, входящие в соответствующее расписание повторения, начиная с даты **startDate**. | Бесконечное повторение события с 1 июня 2017 г. | **type**, **startDate** |
|Диапазон с определенным количеством повторений | `numbered` | Событие повторяется определенное количество раз (указанное в **numberOfOccurrences**) в соответствии с расписанием повторения, начиная с даты **startDate**. | Повторение события 10 раз с 1 июня 2017 г.  | **type**, **startDate**, **numberOfOccurrences** |


## <a name="properties"></a>Свойства

| Свойство     | Тип   |Описание|
|:---------------|:--------|:----------|
|endDate|Date|Дата, с которой перестает применяться расписание повторения. В зависимости от того, каково расписание повторения события, последний экземпляр собрания может приходиться на другую дату. Обязательное, если для **type** задано значение `endDate`.|
|numberOfOccurrences|Int32|Количество повторений события. Обязательное свойство, которое должно быть положительным, если для **type** задано значение `numbered`.|
|recurrenceTimeZone|String |Часовой пояс для свойств **startDate** и **endDate**. Необязательное. Если это свойство не задано, используется часовой пояс события.|
|startDate|Date|Дата, с которой начинает применяться расписание повторения. В зависимости от того, каково расписание повторения события, первый экземпляр собрания может приходиться на эту или более позднюю дату. Должно быть задано то же значение, что и для свойства **start** повторяющегося [события](event.md). Обязательное.|
|type|String|Диапазон повторения. Возможные значения: `endDate`, `noEnd`, `numbered`. Обязательное.|
