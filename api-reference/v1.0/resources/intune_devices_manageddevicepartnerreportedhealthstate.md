# <a name="manageddevicepartnerreportedhealthstate-enum-type"></a>Тип перечисления managedDevicePartnerReportedHealthState

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Доступные состояния работоспособности для API работоспособности устройств
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|0|О состоянии работоспособности устройства еще не сообщалось|
|activated|1|Устройство активировано партнером мобильной защиты от угроз, но о работоспособности еще не сообщалось.|
|deactivated|2|Устройство дезактивировано партнером мобильной защиты от угроз Работоспособность устройства неизвестна.|
|secured|3|Устройство считается защищенным партнером мобильной защиты от угроз.|
|lowSeverity|4|Партнер мобильной защиты от угроз уровень угрозы устройству считает низким.|
|mediumSeverity|5|Партнер мобильной защиты от угроз уровень угрозы устройству считает средним.|
|highSeverity|6|Партнер мобильной защиты от угроз уровень угрозы устройству считает высоким.|
|unresponsive|7|Партнер мобильной защиты от угроз считает, что устройство не отвечает. Работоспособность устройства неизвестна.|
