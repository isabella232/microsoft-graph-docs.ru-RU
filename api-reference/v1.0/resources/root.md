# <a name="root-resource-type"></a>Тип ресурса Root

Аспект **Root** указывает, что объект находится на верхнем уровне иерархии.
Если у этого аспекта имеется значение, не равное null, это указывает, что объект является корневым элементом.
Значение null (или отсутствие значения) указывает, что объект не является корневым.

**Примечание**. В настоящее время этот аспект пуст, но в будущих редакциях API он может быть дополнен другими свойствами.
