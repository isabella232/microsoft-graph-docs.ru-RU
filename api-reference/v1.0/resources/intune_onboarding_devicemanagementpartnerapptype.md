# <a name="devicemanagementpartnerapptype-enum-type"></a>Тип перечисления deviceManagementPartnerAppType

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Тип партнерского приложения.
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|0|Тип партнерского приложения неизвестен.|
|singleTenantApp|1|Партнерское приложение является одним клиентом в AAD.|
|multiTenantApp|2|Партнерское приложение — это несколько клиентов в AAD.|
