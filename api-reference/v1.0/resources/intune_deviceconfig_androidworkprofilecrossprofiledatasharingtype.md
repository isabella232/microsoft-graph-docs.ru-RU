# <a name="androidworkprofilecrossprofiledatasharingtype-enum-type"></a>Тип перечисления androidWorkProfileCrossProfileDataSharingType

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Тип общего доступа к данным "От профиля к профилю" для рабочий профилей Android
## <a name="members"></a>Члены
|Член|Значение|Описание|
|:---|:---|:---|
|deviceDefault|0|Значение устройства по умолчанию, без назначения.|
|preventAny|1|Запрещение любого общего доступа.|
|allowPersonalToWork|2|Разрешение запроса общего доступа от личного профиля к рабочему профилю.|
|noRestrictions|3|Отсутствуют ограничения на общий доступ.|
