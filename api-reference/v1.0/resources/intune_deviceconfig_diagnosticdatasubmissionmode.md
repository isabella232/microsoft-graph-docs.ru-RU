# <a name="diagnosticdatasubmissionmode-enum-type"></a>Тип перечисления diagnosticDataSubmissionMode

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Разрешить устройству отправку данных диагностики и использования данных телеметрии, таких как Watson.
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|userDefined|0|Разрешение пользователю задавать установку.|
|отсутствует|1|Данные телеметрии компонентов операционной системы отправлены не были. Примечание: это значение применимо только к корпоративным и серверным устройствам. Использование этой установки на других устройствах приравнивается к установке значения 1.|
|базовый|2|Отправляются базовые данные телеметрии.|
|расширенный|3|Отправляются расширенные данные телеметрии, включающие данные об использовании и аналитики.|
|полный|4|Отправляются полные данные телеметрии, включающие данные диагностики, как, например, состояние системы.|
