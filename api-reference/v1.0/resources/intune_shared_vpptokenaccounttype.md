# <a name="vpptokenaccounttype-enum-type"></a>Тип перечисления vppTokenAccountType

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Возможные типы маркера Программы корпоративных закупок Apple.
## <a name="members"></a>Члены
|Член|Значение|Описание|
|:---|:---|:---|
|business|0|Маркер Программы корпоративных закупок Apple связан с бизнес-программой.|
|education|1|Маркер Программы корпоративных закупок Apple связан с образовательной программой.|
