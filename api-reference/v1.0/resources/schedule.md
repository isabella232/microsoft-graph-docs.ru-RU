# <a name="schedule-resource-type"></a>Тип ресурса "Расписание"

Пространство имен: microsoft.graph

Коллекция объектов [счедулингграуп](schedulinggroup.md) , объектов [сдвига](shift.md) , объектов [тимеоффреасон](timeoffreason.md) и объектов [тимеофф](timeoff.md) в [команде](../resources/team.md). 

## <a name="methods"></a>Методы

| Метод       | Возвращаемый тип  |Описание|
|:---------------|:--------|:----------|
|[Создание или замена расписания](../api/team-put-schedule.md) | [schedule](schedule.md) | Создайте или замените расписание.|
|[Получение расписания](../api/schedule-get.md) | [schedule](schedule.md) | Получение расписания.|
|[Предоставление общего доступа](../api/schedule-share.md) | Нет | Предоставьте общий доступ к диапазону расписания с участниками расписания.|

## <a name="properties"></a>Свойства
|Имя                   |Тип           |Описание                                                                                                                                      |
|-----------------------|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| id                    |string  |Идентификатор расписания.|
| enabled               |Boolean    | Указывает, включено ли расписание для группы. Обязательно.|
| timeZone              |string  | Указывает часовой пояс, в течение которого Группа расписаний использует формат базы данных. Обязательно.|
| провисионстатус       |оператионстатус    | Состояние подготовки расписания. Возможные значения: `notStarted` ,, `running` `completed` , `failed` . |
| провисионстатускоде   |string  | Дополнительные сведения о том, почему не удалось подготовить расписание. |
| тимеклоккенаблед                  |Boolean  | Указывает, включено ли время для расписания.             |
| опеншифтсенаблед                 |Boolean  | Указывает, включены ли для расписания открытые смены.             | 
| свапшифтсрекуестсенаблед                 |Boolean| Указывает, включены ли запросы замены сдвига для расписания.             |
| оффершифтрекуестсенаблед                 |Boolean  | Указывает, включены ли запросы на смену для расписания.             | 
| тимеоффрекуестсенаблед                    |Boolean | Указывает, включены ли для расписания запросы по времени.             | 



## <a name="relationships"></a>Связи
|Имя                   |Тип           |Описание                                                                                                                                      |
|-----------------------|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| Сдвигает   | Коллекция [shift](shift.md)  | Смены в расписании. |
| тимесофф   |Коллекция [тимеофф](timeoff.md)  | Экземпляры повременных вызовов в расписании. |
| тимеоффреасонс   |Коллекция [тимеоффреасон](timeoffreason.md)  | Набор причин незапланированного времени. |
| счедулингграупс   |Коллекция объектов [schedulingGroup](schedulinggroup.md)  | Логическая группа пользователей в расписании (обычно по роли). |
| опеншифтс   |Коллекция [опеншифт](openshift.md) | Набор смен, открытых в группе планирования, в расписании. |
| воркфорцеинтегратионс   |Коллекция [воркфорцеинтегратион](workforceintegration.md)  | Экземпляр интеграции сотрудников для группы с исходящим трафиком в синхронных уведомлениях об изменениях (для поддерживаемых сущностей). |
| свапшифтчанжерекуестс   |Коллекция [свапшифтсчанжерекуест](swapshiftschangerequest.md)  | Запросы на замену для сдвигов в расписании. |
| опеншифтчанжерекуестс   |Коллекция [опеншифтчанжерекуест](openshiftchangerequest.md)  | Запросы на открытие с открытым сдвигом по расписанию. |
| тимеоффрекуест   |Коллекция [тимеоффрекуест](timeoffrequest.md)  | Время, в течение которого запросы отменяются по расписанию. |
