# <a name="devicemanagementexchangeconnectortype-enum-type"></a>Тип перечисления deviceManagementExchangeConnectorType

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Тип настраиваемого соединителя Exchange.
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|onPremises|0|Подключается к локальной среде Exchange.|
|зарегистрированный|1|Подключается к многопользовательской среде Exchange O365|
|serviceToService|2|Служба Intune подключается непосредственно к многопользовательской среде Exchange O365|
|выделенный|3|Подключается к выделенной среде Exchange O365.|
