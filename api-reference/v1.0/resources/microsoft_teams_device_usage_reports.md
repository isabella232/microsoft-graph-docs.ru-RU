# <a name="microsoft-teams-device-usage-reports"></a>Отчеты об использовании устройств Microsoft Teams

С помощью отчетов об использовании устройств Microsoft Teams можно анализировать использование устройств Microsoft Teams в организации.

## <a name="methods"></a>Методы

| Метод                                   | Возвращаемый тип | Описание                              |
| :--------------------------------------- | :---------- | :--------------------------------------- |
| [Получение сведений о пользователях](../api/reportroot_getteamsdeviceusageuserdetail.md) | Stream      | Получение сведений об использовании устройств Microsoft Teams отдельными пользователями. |
| [Получение количества пользователей](../api/reportroot_getteamsdeviceusageusercounts.md) | Stream      | Получение сведений о количестве уникальных пользователей в день по типам устройств. |
| [Получение распределенного количества пользователей](../api/reportroot_getteamsdeviceusagedistributionusercounts.md) | Stream      | Получение количества уникальных пользователей по типам устройств за выбранный период времени. |
