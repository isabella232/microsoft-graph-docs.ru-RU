# <a name="applockerapplicationcontroltype-enum-type"></a>Тип перечисления appLockerApplicationControlType

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Возможные значения типов управления приложением AppLocker
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|notConfigured|0|Значение устройства по умолчанию, тип управления приложением не выбран.|
|enforceComponentsAndStoreApps|1|Принудительное использование компонента и приложений магазина Windows.|
|auditComponentsAndStoreApps|2|Аудит компонента и приложений магазина Windows.|
|enforceComponentsStoreAppsAndSmartlocker|3|Принудительное применение компонентов, приложений магазина Windows и smart locker.|
|auditComponentsStoreAppsAndSmartlocker|4|Аудит компонентов, приложений магазина Windows и smart locker.|
