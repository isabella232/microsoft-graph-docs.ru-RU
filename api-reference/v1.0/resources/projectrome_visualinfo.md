# <a name="visualinfo-resource-type"></a>Тип ресурса visualInfo

Сложный тип для представления свойства **visualElements** в объекте [activity](../resources/projectrome_activity.md).

Каждое действие пользователя отображаются на временной шкале как адаптивная карточка. Разработчикам приложений рекомендуется предоставить настраиваемую карточку, которая записывает сущность действий, которые выполнялись в вашем приложении. Это можно сделать с указанием настраиваемой карточки JSON в свойстве содержимого.

В дополнение к визуальным метаданным с адаптивной карточкой, приложение может указывать содержимое метаданных — данные, которые используются для формирования вывода действия пользователя, чтобы предложить новые действия для будущего повторного взаимодействия. Это можно осуществить с помощью действия свойства contentInfo для предоставления объекта JSON, который использует свойства schema.org для описания содержимого.

Если этот параметр не указан, простая настраиваемая карточка будет создаваться с использованием отображаемого текста и описания свойств. Рекомендуется использовать настраиваемые карточки для демонстрации наилучшего содержимого в вашем приложении.

## <a name="properties"></a>Свойства

|Имя | Тип | Описание|
|:----|:------|:-----------|
|displayText | String | Обязательно. Короткое текстовое описание уникального действия пользователя (например, имя документа в тех случаях, где действие относится к созданию документов)|
|description | String | Необязательный параметр. Более длинное текстовое описание уникальный активности пользователя (пример: имя документа, первое предложение и/или метаданные)|
|backgroundColor | String | Необязательный параметр. Цвет фона, используемого для преобразования для просмотра активности в пользовательском интерфейсе - цвет торговой марки для источника действия приложения. Должен быть допустим шестнадцатеричный цвет|
|content | Нетипизированный объект JSON | Необязательный параметр. Настраиваемые части данных — объект JSON, используемый для предоставления настраиваемого содержимого для преобразования для просмотра активности в пользовательском интерфейсе оболочки Windows|
|attribution | [imageInfo](../resources/projectrome_imageinfo.md) | Необязательный параметр. Объект JSON, используемый для представления значка, представляющего приложение, используемое для создания действия|
