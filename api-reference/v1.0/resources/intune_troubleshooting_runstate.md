# <a name="runstate-enum-type"></a>Тип перечисления runState

> **Важно:** API под бета-версией в Microsoft Graph - предварительной версии и могут быть изменены. Использование этих API в производственных приложениях не поддерживается.

> **Примечание.** Для настройки элементов управления и политик Intune с помощью API Microsoft Graph по-прежнему требуется, чтобы клиент [лицензировал](https://go.microsoft.com/fwlink/?linkid=839381) Intune надлежащим образом.

Указывает тип состояния выполнения сценария управления устройства.
## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|unknown|0|Неизвестный результат.|
|success|1|Скрипт выполняется успешно.|
|fail|2|Не удалось выполнить сценарий.|
